# 🦀 MangueTown: O Resgate dos Acordes 🎸
### Um Cozy Game sobre o Recife, o Manguebeat e a Lógica de Programação.

---

## 📝 Sobre o Jogo
**MangueTown** é um jogo ambientado no Recife dos anos 90. Você controla, um menino do mangue que, acompanhado por seu caranguejo de estimação, tem a missão de montar uma banda, salvar o manguezal da poluição e realizar o show histórico no Pátio de São Pedro.

O jogo é uma jornada educativa onde o jogador aprende sobre o movimento Manguebeat enquanto aplica conceitos fundamentais de **Estrutura de Dados e Algoritmos**.

---

## 🚀 Como Executar

### Pré-requisitos
1.  Ter um compilador C instalado (Ex: MinGW/GCC).
2.  Ter a biblioteca **Raylib** configurada.

### Passo a Passo
1.  Clone este repositório:
    ```bash
    git clone [https://github.com/vitoriaduran/MangueTown.git](https://github.com/vitoriaduran/MangueTowngit)
    ```
2.  Navegue até a pasta do projeto:
    ```bash
    cd MangueTown
    ```
3.  Compile o jogo usando o Makefile:
    ```bash
    make
    ```
4.  Execute o programa:
    ```bash
    ./MangueTown.exe
    ```

---

## 🕹️ Controles
* **Setas Esquerda/Direita:** Movimentar Zezinho.
* **Espaço:** Interagir com NPCs e coletar itens.
* **Enter:** Confirmar seleções nos menus.
* **M:** Abrir a Mochila (Pilha).
